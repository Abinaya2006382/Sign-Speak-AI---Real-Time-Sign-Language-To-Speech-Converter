# Sign-Speak-AI---Real-Time-Sign-Language-To-Speech-Converter
Communication between hearing-impaired individuals and people who do not understand sign language is difficult. Existing solutions are often expensive, less accessible, or not real-time. This project aims to use AI and computer vision to recognize sign language gestures and convert them into speech instantly, enabling seamless communication.

## Project Description
SignSpeak AI is an AI-powered system that converts sign language gestures into speech in real time. The project uses computer vision and machine learning techniques to recognize hand gestures, convert them into text, and generate speech output. This helps bridge the communication gap between hearing-impaired individuals and people who do not understand sign language.

---

## Objectives
- Detect sign language gestures in real time.
- Convert gestures into meaningful text.
- Convert text into speech output.
- Improve communication accessibility for hearing-impaired individuals.
- Provide a user-friendly and efficient communication tool.

---

## Technologies Used
- Python
- OpenCV
- MediaPipe
- TensorFlow / Keras
- Text-to-Speech (pyttsx3)
- Flask / Streamlit
- SQL Database

---

## Features
- User Registration and Login
- Real-Time Hand Gesture Capture
- Sign Language Detection
- Gesture-to-Text Conversion
- Text-to-Speech Conversion
- Speech Playback
- Prediction History Management
